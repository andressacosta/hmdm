## Installation

cd your hugo site directory and run:

```shell
$ mkdir themes
$ cd themes
$ git clone -b  https://github.com/surajmandalcell/potato-dark
```

## Usage

Use hugo's -t potato-dark or --theme=potato-dark option with hugo commands. Example:

```shell
$ hugo server -t potato-dark -w -D
```
